Reproducibility Notes

Final bag sourceExported from live_camera_calibrated_R_01, the final calibrated Raspberry Pi hardware run.
Static artifactAll UI behavior is plain HTML, CSS, canvas, and local JSON. No package install or network fetch is required.
IMM comparisonMode probabilities expose how the formal tracker weighs smooth and maneuvering motion models.
Dropout evidenceMeasurement age is clamped at zero for display, then rises only when detections are actually stale or absent.
Data source: docs/assets/ghost_live_dashboard/live_camera_calibrated_R_01_dashboard.json. To view locally, serve the repository docs folder with cd docs && python3 -m http.server 8000 --bind 0.0.0.0; direct file opening may block JSON loading in some browsers.
