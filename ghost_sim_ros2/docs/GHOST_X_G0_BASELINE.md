## One-command reproduction

From the repository root:

```bash
bash ghost_sim_ros2/tools/rebuild_ghost_x_baseline.sh
```

This command:

1. regenerates all preserved hardware bag plots;
2. regenerates the machine-readable baseline manifest;
3. verifies all required external evidence exists;
4. verifies hashes and bag metadata are readable;
5. runs the focused mission/world regression tests.

The repository-owned software and deterministic simulation reproduce from a clean checkout. Hardware plot reproduction additionally requires the preserved external bag, calibration file, and physical-session directory listed in the manifest.
